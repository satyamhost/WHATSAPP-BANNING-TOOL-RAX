- Simple menu-based system
- Email automation support
- Temporary & permanent data tracking
- Beginner-friendly interface
- Colorful terminal output

- Python 3
- Internet connection
- Git

#⚙️ Installation Guide

---

#🐧 For Linux Users

1️⃣ Update system

sudo apt update && sudo apt upgrade -y

2️⃣ Install Python & Git

sudo apt install python3 git -y

3️⃣ Clone Repository

git clone https://github.com/satyamhost/WHATSAPP-BANNING-TOOL-RAX.git

4️⃣ Go to folder

cd WHATSAPP-BANNING-TOOL-RAX

5️⃣ Install requirements

pip3 install -r requirements.txt

6️⃣ Run tool

python3 main.py

---

#📱 For Termux Users

1️⃣ Update packages

pkg update && pkg upgrade -y

2️⃣ Install Python & Git

pkg install python git -y

3️⃣ Clone Repository

git clone https://github.com/satyamhost/WHATSAPP-BANNING-TOOL-RAX.git

4️⃣ Open folder

5️⃣ Install requirements

6️⃣ Run tool
